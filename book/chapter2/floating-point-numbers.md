# 浮点数

Go 具有与 [IEEE 754 标准](https://zh.wikipedia.org/wiki/IEEE_754)相符合的浮点数：

* `float32` 是一个 4 字节的浮点数（在 C 语言中称为 `float`）
* `float64` 是一个 8 字节的浮点数（在 C 语言中称为 `double`）

`float32` 和 `float64` 的[零值](zero-values.md)是 0.0。
