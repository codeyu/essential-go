Hello, World!

你好，世界！

创建一个文件 hello_world.go：

用 go run hello_world.go 运行它。

go run 对于单文件程序的临时执行非常有用。

您还可以使用 go build hello_world.go 创建可执行文件。

在 Windows 上，这将创建 .\hello_world.exe 可执行文件。

在其他OS上，这将创建 ./hello_world 可执行文件。
