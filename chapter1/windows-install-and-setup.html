Windows 系统安装和设置

Windows 系统安装和设置

安装

从 https://golang.org/dl/ 下载 msi 安装包安装。

设置

安装后，您需要配置GOPATH环境变量。

从 Go 1.8 开始，GOPATH环境变量具有默认值%USERPROFILE%/go。

您应该了解 GOPATH 在 go 工具中的作用。
